# What is noauth.sh?

**noauth.sh** is a service that drives the future of "passwordless authentication" as a core principle, democratizing access to biometric authentication while delivering a simpler and more secure user experience. With **noauth.sh**, you can boost conversions, streamline user authentication, and reduce the risk of phishing attacks and other credential-based threats.

## Easy Integration
*noauth* provides an easy and flexible way to integrate biometric authentication into your website, application, or existing authentication workflow. The SDK is simple and straightforward, abstracting the complexity of biometric processes while adhering to modern standards like WebAuthn and FIDO2. It's designed to deliver an excellent Developer Experience (DX) through a robust library that lets you choose between two programming paradigms: functional programming (FP) or object-oriented programming (OOP).
